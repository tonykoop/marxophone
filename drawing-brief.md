# Marxophone Drawing Brief

Scaffolded 2026-07-01 during V5 refresh. This L2 packet has **no measured design
table**; no dimensioned drawings exist and none should be released until coupon
data selects real values (design.md "Authority Boundary").

## Required first drawings (after coupon data exists)

- One-course action coupon fixture: string bed segment, bridge/termination,
  interchangeable hammer arm + tip mount, spring-return mount.
- String-bed layout: fretless course plan, bridge line, string plane height —
  all `TBD` until scale/tension/course count are measured.
- Spring-hammer action detail: arm length/pivot, tip contact geometry, spring
  mounting — `TBD` until rebound trials fix an action envelope.
- Zither soundbox outline + soundboard/bracing plan — `TBD` until soundboard
  coupon response and bridge load review exist.

## Authority notes

- `cad/marxophone.scad` is a GENERIC REVIEW ENVELOPE only (prototype_assumption);
  it traces to no measured authority. Its values are massing placeholders, not
  drawing dimensions.
- Keep every drawing row at concept_only / pending_measurement until measured
  templates or reviewed CAD exist. Do not conflate string-bed tuning with
  hammer-action behavior in the same first drawing (design.md L2 Review Checklist).
